// how to dockerize nodejs app
first keep your project ready

and then create Dockerfile

and then build the docker image
docker build -t <image_name> .

ex. docker build -t docker-node-app .


how to see images 
docker images


how to run 
docker run -d -p 3000:3000 --name <container_name> <image_name>

ex: docker run -d -p 3000:3000 --name nodejs-container docker-node-app

-d (detach mode)


how to stop container 
docker stop <container_id or container_name>

how to remove container 
docker rm <container_id or container_name>

how to stop and remove the container ?
docker rm -f <container_id or container_name>

to see all running continers 
docker ps 

how to see logs of docker image 
docker logs <container_id or container_name>

docker logs -f <container_id or container_name> (for realtime logs)
